Per consentire la gestione degli accessi alle risorse é necessario definire i cosidetti "access points".

Un access point é un punto a cui l'utente deve autenticarsi per poter accedere alla risorsa richiesta.

Dal punto di vista dell'utente l'access point si presenta come una pagina web che contiene il modulo per l'autenticazione o come finestra del browser se viene selezionata l'autenticazione HTTP.

Un access point puó essere associato solo ad un nodo e deve necessariamente risiedere la dove il nodo é definito.
