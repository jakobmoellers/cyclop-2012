» HOME » Network
NOTA:

Prima di procedere con questa attivitá é utile leggere la breve panoramica sull'ambiente multiserver nel caso in cui non si utilizzi il prodotto in un ambiente a server singolo.


Se si utilizza il prodotto in un ambiente con un unico server di autenticazione è necessario definire solo il nodo relativo a quest'ultimo:
  1. Cliccare su "Network"
  2. Cliccare su "Nodi"
  3. Cliccare su "Aggiungi Nodo"
  4. Inserire i dati relativi al nodo
Se invece si intende configurare un ambiente con piú server SUMO é indispensabile definire un nodo per ogni server al fine di rendere operativo il sistema.
Seguire quindi i punti di cui sopra, e ripetere l'operazione dal punto 3 per ogni server.

NOTA:

Se si dispone di un server che dispone di piú indirizzi IP, e si desidera avere accesso alle risorse definite su quest'ultimo (access points) sui quale desidera


Il primo é standard ed é definito sull'indirizzo locale (127.0.0.1), ed é finito piú nodi se il sistema deve operare in un ambiente con piú server di autenticazione.
