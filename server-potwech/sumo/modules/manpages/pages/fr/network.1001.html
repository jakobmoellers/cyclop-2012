» HOME » Network » Nodi
Il prodotto consente di gestire un ambiente in cui é necessario utilizzare piú di un server per la gestione degli accessi alle risorse.

Ogni server rappresenta quello che nell'applicazione viene definito "nodo". Ogni nodo é una installazione indipendente del prodotto e possiede una propria console di gestione.
L'insieme di piú nodi costituisce l'ambiente del sistema (detto anche "cella").

Un singolo nodo puó gestire piú access points.


IMPORTANTE:

Poiché é possibile eseguire installazioni multiple del prodotto sullo stesso server é importante sottolineare che:
  • I nodi che non condividono lo stesso database non fanno parte della "cella", e sono estranei alla gestione centralizzata del sistema, poiché una cella puó contenere una sola "console principale".
  • Prestare attenzione alla configurazione del sistema se il database principale é condiviso da piú nodi dello stesso server fisico.
